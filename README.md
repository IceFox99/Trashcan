## 结语
**这些职业和练法算是覆盖了基本所有需要刷的东西，符文、85场景、打钱、底材、6boss、DC、巴尔司机等等都有各自效率最高的角色，刷累了换个角色我又是一条好汉。等这七个练法都成型后就可以开始考虑娱乐玩法了，或者就退坑了。**
